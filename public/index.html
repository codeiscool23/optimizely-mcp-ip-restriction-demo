How to run the demo: 1) On your trusted Wi-Fi, Connect each card and complete the login. 2) Test connection — both go green and load their tool lists. Optionally Call a tool to show live data. 3) Switch your machine to the mobile hotspot, then click ↻ Reload IP — the IP turns amber (baseline stays fixed). 4) Test connection / Call tool again — same tokens, new IP, so a restricted server now returns an auth/forbidden error.
